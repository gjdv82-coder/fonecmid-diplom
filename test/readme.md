# Инструкция для тестирования

----

## Шаг 1
  В базу добавляем пользователей с профилем "Специалист": "Романов А.В." и "Васильев М.В."

  В базе заходим в раздел "Расчет управленческой зарплаты" далее "Условие оплаты сотрудников". 
   Добавляем условия оплаты для пользователей "Романов А.В." и "Васильев М.В.".

----

## Шаг 2
  Для получения обработки выполните следующие действия:
  Открыть https://github.com/Pr-Mex/vanessa-automation.
  Перейти к разделу с релизами — Releases.
  Выбрать релиз.
  В нижнем разделе найти файл с названием vanessa-automation-single.
  Скачать его.
  Обработка запускается с использованием тонкого (толстого) клиента, при этом важно указать в параметрах запуска — /TESTMANAGER. 
  Таким образом  программа будет определена как «Менеджер тестирования».

----

## Шаг 3
  Запускаем внешнюю обработку vanessa-automation.epf. 
  Настраиваем вкладку "Клиенты тестирования" добавляем клиентов тестирования.
  
  заполняем "путь к инфобазе" File="ПутьКБазе";  
  дополнительные параметры /N"ИмяПользователя".
  
  Выбираем соответствующего клиента тестирования и подключаем его для тестировния файла для данного профиля пользователя.

----

## Шаг 4
  Запускаем последовательно файлы для тестирования в Vanessa-automation:
   
  - Пользователь с профилем "Менеджер": 1_Создание_обслуживания_клиента.feature 
	
  - Пользователь с профилем "Специалист": 2_Закрытие_обслуживания_клиентов.feature
  
  - Пользователь с профилем "БухгалтерИТФирмы": 3_Массовое_создание_актов.feature
   
  - Пользователь с профилем "БухгалтерИТФирмы": 4_Формирование_отчета_анализ_выставленных_актов.feature
