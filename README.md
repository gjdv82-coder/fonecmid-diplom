# Инструкция предназначена для настройки доработок. 
    *Реализована схема Сотрудник = Пользователь.

------

## Шаг 1
    Под Админнистраторм заходми в раздел "Администрирование" -> "Настройки пользователей и прав" -> "Профили групп доступа" 
    и добавляем профили:
    - Специалист - обязательно должна быть роль "Работа с документами обслуживания клиентов".
    - Менеджер - обязательно должна быть роль "Работа с документами обслуживания клиентов" и "Отправка сообщений в Телеграм".
    - БухгалтерИТФирмы - обязательно должна быть роль "Массовое создание документов реализации".
    - КадровикРасчетчик - обязательно должна быть роль "Расчет зарплаты".

------

## Шаг 2
    Далее заходим в раздел "Администрирование" -> "Настройки пользователей и прав" -> "Пользователи" и добавляем пользователей.
    В вкладке права доступа указываем соответствующие профили.

------

## Шаг 3 
    Реализована возможность оповещения специалистов через Телеграм. Для правильной работы оповещения производим настройки:
    
    Создание бота
    Напишете в Телеграм https://t.me/BotFather команду "/start"
    Напишите команду "/newbot"
    Выберите имя вашего бота, которое будут видеть пользователи
    Выберите идентификатор вашего бота (он должен заканчиваться на "bot")
    Скопируйте токен, который пришлет BotFather и сохраните его В разделе "Работа с заявками клиентов" -> "Токен управления телеграм-ботом"
    
    Создание группы
    Создайте группу в Телеграм
    Добавьте в группу только что созданного бота
    Назначьте боту права администратора
    Получите идентификатор группы
    Напишите любое сообщение в группу
    С помощью браузера или Postman выполните запрос https://api.telegram.org/bot[ВашТокен]/getUpdates. В полученном JSON найдите идентификатор группы, в которой получено сообщение, и сохраните его в разделе "Работа с заявками клиентов" -> "Идентификаторе группы для оповещения".

    Для автоматической отправки сообщений в Телеграм.
    Заходим в раздел "Администрирование" -> "Обслуживание" -> "Регламентные и фоновые задания" включаем регламентное задание "Обработка уведомлений".
    Производим настройки периодичности (рекомендую периодичность делать больше 5 минут).

    Для ручного оповещения можно написать сообщение в разделе "Работа с заявками клиентов" -> "Уведомление телеграм-боту" и отправить через "Отправка сообщения телеграм".

------

## Шаг 4
    В разделе "Работа с заявками клиентов" добавляем номенклатуру в "Номенклатура абонентская плата" и "Номенклатура работы специалиста".

------

## Шаг 5
    Настройка раздела "Расчет управленческой зарплаты".
    - Добавляем минимальный оклад в подразделе "Сервис" -> "Минимальный оклад".
    - Добавляем Графики работ в подразделе "Справочники" -> "Графики работ" (для расчета дней отпуска добавте график "Календарный").
    - Заполняем графики работ через процедуру в подразделе "Сервис" -> "Заполнение графика" (для графика "Календарный" выходные дни оставляем пустым).
    - В подразделе "Операции" -> "Условия оплаты сотрудников" добавляем сведения на сотрудников.
